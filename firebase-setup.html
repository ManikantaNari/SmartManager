Firebase Setup Guide

One-time setup for Smart Manager (Free Forever)

Estimated Time: 10-15 minutes
1
Create Google Account (Skip if you have one)

If you already have a Gmail account, skip to Step 2.

Create Google Account
2
Go to Firebase Console

Click the button below and sign in with your Google account:

Open Firebase Console
3
Create New Project
  1. Click "Create a project" or "Add project"
  2. Enter project name: smart-manager (or any name you like)
  3. Click Continue
  4. Disable Google Analytics (not needed) ? Click Continue
  5. Wait for project to be created ? Click Continue
4
Add Web App
  1. On the project page, click the Web icon (</>)
  2. Enter app nickname: Smart Manager
  3. Don't check "Firebase Hosting"
  4. Click Register app
  5. You'll see a code block with firebaseConfig - KEEP THIS PAGE OPEN!
Important: Don't close this page! You'll need to copy the config values in Step 7.
5
Enable Firestore Database
  1. In the left sidebar, click Build ? Firestore Database
  2. Click Create database
  3. Select "Start in test mode" (we'll secure it later)
  4. Choose location: asia-south1 (Mumbai) for best speed in India
  5. Click Enable
6
Enable Authentication
  1. In the left sidebar, click Build ? Authentication
  2. Click Get started
  3. Click Anonymous (under "Native providers")
  4. Toggle Enable to ON
  5. Click Save
Why Anonymous? This allows the app to work without login. Your data is still private and synced!
7
Copy Your Firebase Config

Go back to your Firebase project settings:

  1. Click the gear icon (Settings) next to "Project Overview"
  2. Click Project settings
  3. Scroll down to "Your apps" section
  4. Find the firebaseConfig code block
  5. Copy each value and paste below:

Your config is ready!

Next: Open smart-manager.html in a text editor, find the firebaseConfig section at the top, and replace it with this config.

8
You're Done!
